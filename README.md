# go-token-introspection

a typed client and response model for RFC 7662 OAuth 2.0 Token Introspection.

Implements **RFC 7662 — OAuth 2.0 Token Introspection** (Proposed Standard (RFC 7662), 2015-10).
Spec: <https://www.rfc-editor.org/rfc/rfc7662.html>

## Status

Bootstrap / pre-v0.1.0. The public API is not yet stable.

## Install

```bash
go get github.com/hstern/go-token-introspection
```
